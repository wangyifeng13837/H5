常用属性
属性 用法 含义
placeholder input中 vaule= 换成 placeholder= 占位符，当用户输入时，里面的文字消失，删除文字时文字返回输入框
autofocus input中text类型 属性位置添加autofocus即可 规定当页面加载时input元素自动获得焦点
multiple input file类型 属性为multiple 多文件上传
autocomplete 1.autocomplete 首先需要提交按钮 2.这个表单必须要有name
input text类型中 添加complete属性，默认为on 		规定表单是否应该启用自动记录完成功能，on表示记录，off表示不记录
required input 添加required属性 必填项
accesskey input 添加accesskey属性 规定激活元素的快捷键 采用alt+字母的形式
